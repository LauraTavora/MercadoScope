# Como apresentar este projeto no portfólio

## Título sugerido

**MercadoScope - Automação de coleta e análise de preços para e-commerce**

## Descrição curta

Aplicação web que automatiza a pesquisa de produtos, organiza preços e sinais de reputação, salva resultados em banco de dados e gera CSV e relatório PDF. O projeto foi criado com FastAPI, SQLAlchemy, Playwright e uma interface responsiva.

## Problema resolvido

A comparação manual de anúncios é demorada, difícil de repetir e pouco rastreável. O sistema transforma a atividade em jobs com histórico, progresso, limites por plano e exportações padronizadas.

## Destaques técnicos

- API REST versionada.
- Arquitetura multi-tenant com planos nativos.
- Provedores mock, browser e API.
- Coleta em segundo plano.
- CSV compatível com Excel.
- Relatório PDF automático.
- Docker, CI, testes e documentação.
- Onboarding e versão visível.

## Demonstração recomendada

1. Inicie no modo `mock`.
2. Pesquise por `notebook gamer`.
3. Mostre o progresso da automação.
4. Apresente o gráfico e a tabela.
5. Baixe o CSV e o PDF.
6. Abra `/docs` e mostre os contratos da API.

## Próximos passos comerciais

Agendamento recorrente, alertas de alteração, integração com outros marketplaces, dashboard histórico e versão white-label para agências.
