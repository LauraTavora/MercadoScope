# Onboarding e Product Tour

## Primeira visita

Um modal apresenta o fluxo em quatro passos. O estado é salvo em `localStorage`.

## Fluxo recomendado

1. Começar no provedor `mock`.
2. Pesquisar “notebook gamer”.
3. Acompanhar o progresso.
4. Explorar gráfico e tabela.
5. Baixar CSV e PDF.
6. Ler `/docs` para testar a API.

## Ativação de dados reais

- Validar regras do site e uso permitido.
- Preferir a API oficial para uso contínuo.
- Configurar variáveis de ambiente.
- Executar um job pequeno.
- Revisar logs, dados e campos nulos.
